Privacy Policy

Live Closed Captions — last updated February 2026

Overview

Live Closed Captions is a Chrome extension that displays real-time speech-to-text captions on any browser tab. The extension itself collects no personal data, makes no network requests of its own, and stores no transcribed text.

What data is collected

The extension does not collect, transmit, or store any personal information.

The only data persisted locally is your UI preferences (caption bar position and font size), stored in chrome.storage.sync. This contains no personally identifiable information.

Speech recognition and audio

Captions are powered by the browser's built-in Web Speech API (webkitSpeechRecognition). When captions are active, Chrome streams raw audio from your microphone to Google's speech recognition servers and returns transcribed text. This is the same pipeline used by Google Chrome voice search and Google Slides' built-in captions.

The extension has no access to the audio stream itself and receives only the transcribed text strings returned by the API. Transcribed text is held in memory only and is discarded after 5 seconds of silence or when captions are stopped. It is never written to disk, sent anywhere, or logged.

Google's use of audio data through the Web Speech API is governed by Google's Privacy Policy.

Permissions

The extension requests the following Chrome permissions:

  • Microphone — required for speech recognition. Audio is processed by Chrome's Web Speech API; the extension does not record or store audio.
  • Tabs — used to manage the speech recognition tab and deliver captions to all open tabs.
  • Scripting — used to inject the caption overlay into web pages.
  • Storage — used to save your position and font size preferences.
  • Host permissions (<all_urls>) — required to display the caption overlay on any website you visit.

Third parties

Other than Google's Web Speech API (invoked by Chrome itself), the extension communicates with no third-party services. There is no analytics, no advertising, no crash reporting, and no usage tracking of any kind.

Children

This extension does not knowingly collect any information from anyone, including children.
